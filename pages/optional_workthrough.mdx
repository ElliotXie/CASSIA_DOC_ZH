# 可选智能体介绍

感谢你阅读到这里！基本CASSIA工作流在大部分的情况下已经可以很好地完成任务。为了应对一些比较特殊的情况，接下来我们介绍一些高阶的智能体。


## 不确定性量化智能体 （UQ Agent）

是大模型就有可能会犯错；如果我们假设大模型每次回答正确的概率是80%，那么让大模型回答同一个问题五次，取他们中出现次数最多的回答，如此这般大模型答对这个问题的概率理论上会是94.2%!

UQ agent 主要提供这样的功能，让CASSIA的回答更加准确可靠。

## 注释增强智能体 （Annotation Boost Agent)

这个智能体在前面的工作流中出现过，但你也可以选择单独将它用在你选择的簇上面。注释增强智能体可以阅读CASSIA的注释报告，然后通过不断地生成假设，并且调取簇的基因表达信息来验证假设，最终优化注释的结果。在我们的测试中，该智能体的表现非常出色，也是CASSIA的核心创新之一。

## 判官智能体 （Comparecelltype Agent)

我们知道基本CASSIA工作流会输出三个可能的最终注释，并以可能性排列从大到小排列。有的时候我们会遇到模棱两可的注释，这个时候可以使用判官智能体来找出更有可能的那个注释结果。该智能体实际上是n个智能体的组合，每个智能体会分别为可能的注释结果打分。根据得分情况选择最后最有可能的注释结果。

## 亚聚类分析智能体 （Subclustering Agent)

一次分群往往是不够的，我们会有一些特别感兴趣的分群需要被提取出来后进行再一次的分析，亚聚类分析智能体可以同时比较多个分群，给出更准确的且细致的注释。

## 注释增强智能体+ （Annotation Boost Agent +)

该智能体是注释增强智能体的进一步升级，可以执行针对某个簇的个性分析，目前仍在持续升级开发中，请敬请期待...

## 知识检索智能体 （RAG Agent）

该智能体是CASSIA的核心创新之一，根据用户提供的基本信息，它会自动检索marker数据库，自动检索cell ontology tree， 并且一种在细胞种类上进行PCA的方式，得出一系列背景信息来辅助基本CASSIA工作流，在需要非常细致或者新颖的注释的时候，该智能体显著提升CASSIA的表现。

由于库的依赖问题，知识检索智能体目前只能在Python中使用，我们也会尽快推出r的版本。
